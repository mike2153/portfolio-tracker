---

## v2 API Characteristics:
1. All responses use standardized v2 format with metadata
2. All data is permanently stored in Supabase
3. External APIs (Alpha Vantage) only called when data is missing/stale
4. Pagination supported on all list endpoints
5. Numeric values returned as strings for precision
6. Comprehensive error responses with details array
7. Cache status and data source included in metadata
